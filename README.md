## 🚀 How to Deploy (GitHub Pages)

1. Create a new repo on GitHub
2. Upload `index.html` to the repo
3. Go to **Settings → Pages**
4. Set source: `Deploy from a branch` → `main` → `/ (root)`
5. Save → your site will be live in ~60 seconds
